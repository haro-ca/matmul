Shared Memory Tiling
CUDA Kernel Code
TILE = 16 @cuda.jit def matmul_tiled_v1(C, A, B): sA = cuda.shared.array((TILE, TILE), dtype=float32) sB = cuda.shared.array((TILE, TILE), dtype=float32) tx = cuda.threadIdx.x # columna local ty = cuda.threadIdx.y # fila local col = cuda.blockIdx.x * TILE + tx row = cuda.blockIdx.y * TILE + ty _, K = A.shape acc = 0.0 for t in range(K // TILE): # asume K múltiplo de TILE sA[ty, tx] = A[row, t * TILE + tx] sB[ty, tx] = B[t * TILE + ty, col] cuda.syncthreads() # SYNC 1: todos cargaron for k in range(TILE): acc += sA[ty, k] * sB[k, tx] cuda.syncthreads() # SYNC 2: nadie sobrescribe aún C[row, col] = acc
Visual Explanation
📦 Shared Memory Allocation
sA[16][16]

Each thread block allocates two 16×16 shared memory arrays for fast data reuse
🧵 Thread Indexing
tx = threadIdx.x (0-15)
→
ty = threadIdx.y (0-15)
→
Global position
col = blockIdx.x × 16 + tx
row = blockIdx.y × 16 + ty
🔄 Tile Loading Loop
for t in range(K // 16)
→
Load tile t
Iterates over tiles along the K dimension. For 1024×1024 matrices with K=1024: 64 iterations
🤝 Cooperative Loading
Thread (0,0)
loads sA[0,0]
Thread (1,0)
loads sA[0,1]
Thread (0,1)
loads sA[1,0]
Thread (1,1)
loads sA[1,1]
...
Thread (15,15)
loads sA[15,15]
256 threads per block cooperatively load 256 elements into each shared memory array
⏸️ Synchronization Points
SYNC 1: After loading - ensures all threads have loaded their data before computation
SYNC 2: After computation - ensures no thread overwrites shared memory while others still reading
⚡ Computation Phase
for k in range(16):
    acc += sA[ty, k] * sB[k, tx]
Each thread computes its dot product using cached shared memory data (16 multiply-adds)
